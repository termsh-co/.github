# termsh

**Open-source SSH terminal** with a **zero-knowledge vault** and **end-to-end encrypted** cloud sync.

The sync server never sees your master password or plaintext credentials.

🔐 **Zero-knowledge** · **E2EE sync** · **Self-hostable** · **GPL-3.0**
